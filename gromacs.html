GROMACS Molecular Dynamics Manual

This manual describes the complete workflow for running Molecular Dynamics (MD) simulations in GROMACS starting from a molecule.xyz file.

1. Software Requirements

  • GROMACS
  • ACPYPE
  • AmberTools
  • OpenBabel (optional)

2. Starting Structure

Example XYZ file:

5
Methane
C   0.000   0.000   0.000
H   0.629   0.629   0.629
H  -0.629  -0.629   0.629
H   0.629  -0.629  -0.629
H  -0.629   0.629  -0.629

3. Convert XYZ to PDB

obabel molecule.xyz -O molecule.pdb

4. Generate Topology Using ACPYPE

acpype -i molecule.pdb -b MOL

This creates:

MOL.acpype/
File Description
MOL_GMX.gro Coordinate file
MOL_GMX.top Topology file
posre_MOL.itp Position restraints

5. Enter Working Directory

cd MOL.acpype

6. Define Simulation Box

gmx editconf -f MOL_GMX.gro -o boxed.gro -c -d 1.0 -bt cubic
  • -c : center molecule
  • -d 1.0 : 1 nm distance from box edge
  • -bt cubic : cubic box

7. Solvate the System

gmx solvate -cp boxed.gro -cs spc216.gro -o solvated.gro -p MOL_GMX.top

8. Prepare Ion Addition

Create ions.mdp

integrator = steep
nsteps = 500
emtol = 1000.0
emstep = 0.01
cutoff-scheme = Verlet
nstlist = 10
coulombtype = PME
rcoulomb = 1.0
rvdw = 1.0
pbc = xyz

Generate ions.tpr

gmx grompp -f ions.mdp -c solvated.gro -p MOL_GMX.top -o ions.tpr

9. Add Counter Ions

gmx genion -s ions.tpr -o solv_ions.gro -p MOL_GMX.top -pname NA -nname CL -neutral

Select:

SOL

10. Energy Minimization

Create em.mdp

integrator      = steep
emtol           = 1000.0
emstep          = 0.01
nsteps          = 50000

cutoff-scheme   = Verlet
nstlist         = 10

coulombtype     = PME
rcoulomb        = 1.0
rvdw            = 1.0

pbc             = xyz

Run Energy Minimization

gmx grompp -f em.mdp -c solv_ions.gro -p MOL_GMX.top -o em.tpr

gmx mdrun -deffnm em

11. NVT Equilibration

Create nvt.mdp

integrator      = md
dt              = 0.001
nsteps          = 50000

nstxout         = 1000
nstvout         = 1000
nstenergy       = 1000
nstlog          = 1000

cutoff-scheme   = Verlet
nstlist         = 10

coulombtype     = PME
rcoulomb        = 1.0
rvdw            = 1.0

tcoupl          = V-rescale
tc-grps         = System
tau_t           = 0.1
ref_t           = 300

pcoupl          = no

constraints     = h-bonds
constraint_algorithm = lincs

pbc             = xyz

gen_vel         = yes
gen_temp        = 300
gen_seed        = -1

Run NVT

gmx grompp -f nvt.mdp -c em.gro -r em.gro -p MOL_GMX.top -o nvt.tpr

gmx mdrun -deffnm nvt

12. NPT Equilibration

Create npt.mdp

integrator      = md
dt              = 0.001
nsteps          = 50000

nstxout         = 1000
nstvout         = 1000
nstenergy       = 1000
nstlog          = 1000

cutoff-scheme   = Verlet
nstlist         = 10

coulombtype     = PME
rcoulomb        = 1.0
rvdw            = 1.0

tcoupl          = V-rescale
tc-grps         = System
tau_t           = 0.1
ref_t           = 300

pcoupl          = Parrinello-Rahman
pcoupltype      = isotropic
tau_p           = 2.0
ref_p           = 1.0
compressibility = 4.5e-5

constraints     = h-bonds
constraint_algorithm = lincs

pbc             = xyz

gen_vel         = no

Run NPT

gmx grompp -f npt.mdp -c nvt.gro -r nvt.gro -t nvt.cpt -p MOL_GMX.top -o npt.tpr

gmx mdrun -deffnm npt

13. Production MD

Create md.mdp

integrator      = md
dt              = 0.002
nsteps          = 500000

nstxout-compressed = 1000
nstenergy       = 1000
nstlog          = 1000

cutoff-scheme   = Verlet
nstlist         = 20

coulombtype     = PME
rcoulomb        = 1.0
rvdw            = 1.0

tcoupl          = V-rescale
tc-grps         = System
tau_t           = 0.1
ref_t           = 300

pcoupl          = Parrinello-Rahman
pcoupltype      = isotropic
tau_p           = 2.0
ref_p           = 1.0
compressibility = 4.5e-5

constraints     = h-bonds
constraint_algorithm = lincs

pbc             = xyz

gen_vel         = no

Run Production MD

gmx grompp -f md.mdp -c npt.gro -t npt.cpt -p MOL_GMX.top -o md.tpr

gmx mdrun -deffnm md

14. Important Output Files

File Description
.gro Coordinates
.trr Full trajectory
.xtc Compressed trajectory
.edr Energy file
.log Simulation log
.cpt Checkpoint file
.tpr Portable run input

15. Basic Analysis

RMSD

gmx rms -s md.tpr -f md.xtc -o rmsd.xvg

Radius of Gyration

gmx gyrate -s md.tpr -f md.xtc -o gyrate.xvg

Energy Analysis

gmx energy -f md.edr -o energy.xvg
